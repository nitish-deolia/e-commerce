# User Stories - E-Commerce Microservices Platform

This folder contains all user stories organized by microservice and user type.

## Organization Structure
- **01-auth-service**: Authentication and user management stories
- **02-product-service**: Product catalog and browsing stories
- **03-order-service**: Order placement and management stories
- **04-inventory-service**: Stock tracking and management stories
- **05-cross-service**: Stories involving multiple services
- **06-database-setup**: Local PostgreSQL and Liquibase setup stories

## User Types
- **Customer**: End-user browsing and purchasing products
- **Admin**: Managing products, inventory, and system
- **System**: Internal system operations
- **Developer**: Running the platform locally and verifying infrastructure
