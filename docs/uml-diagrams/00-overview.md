# UML Diagrams - E-Commerce Microservices

This folder contains all UML diagrams for the e-commerce platform.

## Structure
- **00-overview.md**: Overview and legend
- **01-class-diagrams.md**: Entity classes and object structures
- **02-sequence-diagrams.md**: Interaction flows between services
- **03-component-diagrams.md**: System architecture and dependencies

## Diagram Types

### Class Diagrams
Show entity relationships within each microservice:
- Auth Service: User, Role, Token
- Product Service: Product, Category, Review
- Order Service: Order, OrderItem
- Inventory Service: Stock, StockHistory

### Sequence Diagrams
Show interactions during key flows:
- User Registration Flow
- Order Placement Flow
- Stock Management Flow

### Component Diagrams
Show high-level system architecture and service dependencies
